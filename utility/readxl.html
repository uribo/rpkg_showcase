readxl: Read Excel Files

エクセルファイルを読み込む

  • CRAN: http://cran.r-project.org/web/packages/readxl/index.html
  • GitHub: http://github.com/hadley/readxl
> library(readxl)

バージョン: 0.1.0.9000

関数名 概略
excel_sheets List all sheets in an excel spreadsheet.
read_excel Read xls and xlsx files.

read_excel

.xls, .xlsx形式のファイルを読み込む。特定のシートを読み込む（初期値は１）場合はsheet引数で指定する

> read_excel(path, sheet = 1, col_names = TRUE, col_types = NULL, na = "", skip = 0)

> datasets <- system.file("extdata/datasets.xlsx", package = "readxl")
